3Q Hatchery · 自動化工具

Facebook 頁面授權

自動從 Graph API Explorer 取得頁面永久 Token，
一鍵部署到 Cloudflare Worker。

STEP 01

開啟 Graph API Explorer

點下方按鈕開啟官方工具，會在新分頁開啟。

開啟 Graph API Explorer ↗
STEP 02

選擇 App 並授予權限

在 Explorer 右側面板：

  • Meta App → 選 3Q Hatchery（App ID: 956987317313843）
  • 點 Generate Access Token
  • 確認勾選以下所有權限再按確認：
pages_manage_posts pages_read_engagement pages_show_list publish_to_groups
STEP 03

複製短效 User Access Token

授權後 Explorer 頂部會出現一段 Token，複製整段（通常以 EAA 開頭）貼在下方：

✓ 頁面 Token 取得成功

PAGE ACCESS TOKEN（永久有效）
請複製後貼到 GitHub Workflow 的輸入框
STEP 04

觸發 GitHub Actions 部署

把上方 Page Token 貼到 Workflow 的 fb_short_token 欄位並執行：

Actions → Setup Facebook Page Token → Run workflow

Workflow 會自動把 Token 部署到 Cloudflare Worker，並存入 KV 永久生效。

開啟 Workflow ↗

完成後可繼續執行 Seed Content Queue 把 14 天素材填入 D1，等下次 cron 自動發文。

填入素材 (Seed Content Queue) ↗
等待 Token 輸入
