# 一站式裝修統包 — 資料中心

> 業內頂價、全程透明、從毛胚到入住一手交付的室內裝修統包。
> 賣的不是最便宜的裝修，是「乾淨、不留隱患、又不被坑」的安心。

**先開這個 →** [`index.html`](index.html)（視覺化導覽中心，分三層、機密層有警示）

---

## ⚠️ 三層分流原則

| 層級 | 能不能公開 | 內容 |
|------|-----------|------|
| 🟢 **公開 · 客戶端** | ✅ 可上官網 | 不含定價邏輯、毛利、獲客手法 |
| ⚪ **工班端** | 內部 | 招募、規範、驗收、合約 |
| 🔴 **內部機密** | ❌ 切勿公開 | 定價毛利、警衛地推、社區滲透等底牌 |

> 官網（GitHub Pages）是公開的。**機密層只留在 repo，永遠不要併進公開官網或外流。**

## 核心邏輯一句話

> **價高 · 質好 · 客傳客** —— 高價養得起好工班 → 好品質 → 客戶信任 → 好評＋轉介 → 更多高價案，圈子越轉越快。

## 怎麼看 HTML

- 直接用瀏覽器開啟。
- 簡報用 `← →` 換頁、`F` 全螢幕、`P` 列印。
- 報價單／保固卡／承諾書右上角有「列印 / 存 PDF」鈕（A4）。
