  3. 工期: 自 ____年__月__日起,至 ____年__月__日止;實際以節點進度為準。
  4. 圖說、材料規格、數量以附件為準;乙方不得擅自降規、換料或變更工法,變更須經甲方書面同意。

三、報價與分期付款(按進度節點,不按天)

  1. 總承攬價金: 新台幣 ____________ 元整(含稅／未稅:______)。

  2. 付款依進度節點分期撥付,各節點須通過甲方(及／或客戶)驗收後始得請款:

    期別 進度節點 比例 金額 驗收條件
    第一期 簽約／進場 __% 完成進場登記與保護
    第二期 隱蔽工程完成複驗 __% 水電／防水複驗通過
    第三期 面材／櫃體完成 __% 分項驗收通過
    尾款 清潔交屋驗收通過 __% 交屋總驗收＋客戶簽收

  3. 各期款於對應節點驗收通過後 ____ 日內給付。

  4. 甲方承諾按時付款;乙方據實完工、配合驗收,雙方互不拖延。

  5. 已扣留之違約金、賠償、改善成本,得自應付款項中抵扣(詳第六條)。

四、完工標準與驗收

  1. 乙方之施工品質須符合《分項施工驗收清單》(本合約附件)所載各項標準。

    驗收標準詳見:工班-施工驗收清單.md

  2. 隱蔽工程(水電、防水等)須經甲方複驗、拍照留存後始得封板;未經複驗擅自封板者,甲方得要求拆除重驗,費用由乙方負擔。
  3. 驗收未通過項目,乙方應於甲方指定期限內無償改善並複驗。
  4. 交屋總驗收經客戶簽收後,視為該案完工。

五、保固責任

  1. 保固期間: 自交屋驗收完成之日起 ____ 年(隱蔽工程之防水、管線等,保固 ____ 年)。
  2. 保固期內因施工瑕疵所致之問題,乙方應於接獲通知後 ____ 日內無償到場修復。
  3. 乙方逾期未修復者,甲方得自行或另委他人修復,費用由乙方負擔或自保固保留款／未付款項中扣抵。
  4. 非可歸責於乙方之事由(正常耗損、人為破壞、不可抗力等)不在此限。
  5. 甲方得於尾款中保留 ____%（保固保留款),於保固期滿無瑕疵後給付。

六、違約與扣款

  1. 乙方違反《工班合作公約與工地行為 SOP》(本合約附件)者,甲方得依其違規處置階梯處理:口頭警告 → 書面警告＋扣款 → 停止派工 → 即刻終止。
  2. 下列情形甲方得自應付款項中扣款並求償:工地整潔不合格、收尾不確實、噪音／作息超時、損壞公共設施或鄰戶、降規偷料、隱匿事故等,扣款金額為實際賠償＋改善成本(或約定之違約金額)。
  3. 零容忍即刻終止事由(無補正機會):工地或社區內抽菸、隨地便溺、辱罵客戶或鄰居、偷竊、私接客戶、破壞透明監看設備、隱匿重大事故。
  4. 乙方無故停工、延誤或擅自退場致甲方受損者,應賠償甲方因此增加之費用及損失。
  5. 終止合作時,已完成且驗收合格之工項按比例結算;未完成或不合格部分不予計價,並扣除善後與另行發包之差額。

七、社區公約遵守條款

  1. 乙方確認已詳閱並同意《工班合作公約與工地行為 SOP》,並使其所帶領之全部人員遵守。

    詳見:工班-合作公約SOP.md

  2. 乙方須遵守各案場社區之施工規範,包含但不限於:全程禁菸、指定如廁、噪音與作息時段、下午約兩三點前撤場、電梯與公共區保護、不擾鄰。
  3. 乙方同意接受全程影像連線監看(透明施工),不得遮蔽、移動或破壞監看設備。
  4. 因乙方人員違反社區規範致甲方遭社區投訴、處罰或喪失案源者,乙方應負賠償責任。

八、保密與不私接客戶條款

  1. 乙方對於施工過程中知悉之客戶資料、案場資訊、報價、甲方營業秘密,負保密義務,非經甲方書面同意不得洩漏或用於本案以外目的。
  2. 乙方不得私下接洽、承攬甲方客戶之本案或衍生工程,亦不得透過甲方案場招攬客戶。
  3. 本條於合約終止後 ____ 年內持續有效。
  4. 乙方違反本條者,應給付甲方懲罰性違約金新台幣 ____________ 元,並賠償甲方所受損害。

九、其他約定

  1. 保險與責任: 乙方應自行為其人員投保必要之勞工保險／意外險;施工期間之工安由乙方負責,因乙方人員所致之人身或財產損害由乙方負責。
  2. 轉包限制: 非經甲方書面同意,乙方不得將承攬工項轉包他人。
  3. 不可抗力: 因天災、政府命令等不可抗力致工期延誤,雙方協商順延,互不負違約責任。
  4. 爭議處理: 本合約之爭議,雙方應先誠信協商;協商不成,合意以 ________ 地方法院為第一審管轄法院。
  5. 附件: 本合約之附件(《合作公約 SOP》、《施工驗收清單》、設計圖說、報價單)為本合約之一部分,與本文具同等效力。
  6. 本合約一式二份,雙方各執一份為憑。
