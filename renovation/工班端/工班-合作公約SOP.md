# 工班合作公約與工地行為 SOP

> 本文件為 3Q 室內裝修統包的工班合作基本規範,適用於所有進場師傅、班頭與其帶領之人員。
> **可作為《按案結算合作合約》之附件,簽署即視為同意全部條款。**
> 一句話精神:**「你顧好品質,案子、收款、客戶我全包。」** 相對的,工地的乾淨、規矩與信任,你要替我顧好。

---

## 〇、為什麼有這份公約(請先讀懂)

我們賣的不是最便宜的裝修,是「**乾淨、不留隱患、不被坑**」的安心。
我們主攻**高端社區與整片新建案**——這類社區規矩特別嚴:不准在外抽菸、下午兩三點就要先撤、不能擾鄰。

> 我們之所以進得去這些好社區,靠的就是**乾淨工地、規範工班、透明施工**。
> 你守好這份公約,就是我們的進場門票;守不好,整個社區都會把我們擋在門外。

對工班而言,這份公約換來的是:**穩定派工、按案結算準時付款、奧客我擋、長期合作。**

---

## 一、進場規範

- [ ] **報到登記**:進場前向案場負責人(工務)報到,確認當日工項、人數、進度節點。
- [ ] **證件與門禁**:依社區規定辦理人員登記、車輛通行證、施工許可單;隨身攜帶可供查驗。
- [ ] **施工許可時間**:嚴格遵守社區公告之可施工時段,未取得許可不得擅自進場或施工。
- [ ] **材料進場**:大型建材、廢料統一走貨梯／指定動線,不得佔用住戶電梯與大廳。
- [ ] **水電源借用**:用水用電依社區規定,不得私接、不得影響鄰戶。
- [ ] **鑰匙與門禁卡**:由公司統一保管與交付,離場確實上鎖,不得轉交無關人員。
- [ ] **每日回報**:當日完工進度、明日工項,於指定群組回報工務(含現場照片)。

## 二、社區友善守則(高端社區的命脈,違反零容忍)

### 2.1 禁菸與如廁
- [ ] **全程禁菸**:工地內、社區公共區域、社區門口、騎樓一律禁菸。要抽菸請離開社區範圍。
- [ ] **嚴禁亂吐檳榔汁、亂彈菸蒂、亂尿尿**——一經發現,**立即停止派工**,無例外。
- [ ] **如廁**:使用指定流動廁所或社區許可之廁所;保持清潔,用完即沖、不留髒污。

### 2.2 噪音與作息時間
- [ ] **可施工時段**:依社區公告為準(常見為平日上午 9:00 起)。
- [ ] **重噪音工項**(打石、切割、敲牆等)集中於社區允許的時段,避開午休。
- [ ] **下午撤場**:多數高端社區要求**下午兩、三點即須收尾撤場**——以該案場公告為準,提前 15 分鐘開始收拾,準時離場,不得拖延。
- [ ] **不得擾鄰**:大聲喧嘩、播放音樂、嬉鬧、在公共區域逗留聊天,一律禁止。

### 2.3 電梯與公共區保護
- [ ] **進場前鋪設保護**:電梯轎廂、地坪、牆面、大廳動線依社區要求做好防護(保護毯／瓦楞板／養生膠帶)。
- [ ] **不踩踏、不刮傷**公共設施;搬運重物以推車並墊護角。
- [ ] **公共區零殘留**:不在梯廳、走道、停車場堆料、調漆、切割、用餐。
- [ ] **離場復原**:當日施工結束,公共區保護視情況留置或拆除,地面清掃乾淨。

### 2.4 對鄰居與管理單位
- [ ] 遇住戶、管委會、警衛,一律**禮貌問好、態度客氣**;有疑問請對方聯繫公司窗口,工班不自行解釋或承諾。
- [ ] 不與鄰居起爭執;任何抱怨第一時間回報工務,由公司出面處理。

## 三、每日工地整潔標準

### 3.1 施工中
- [ ] 工具、材料定位擺放,動線淨空,不隨地散落。
- [ ] 粉塵作業做好防塵(隔離、集塵、灑水);避免粉塵飄入鄰戶或公共區。
- [ ] 用餐集中於指定區域,垃圾不落地。

### 3.2 收尾清掃(每日離場前必做)
- [ ] 地面清掃／吸塵,無明顯粉塵、木屑、釘子、雜物。
- [ ] 工具歸位、電線收齊、危險物品(刀片、釘槍、化學品)收好。
- [ ] 關水、關電、關窗、上鎖,確認無火源與漏水風險。
- [ ] 拍攝「離場照」回報工務,作為透明施工紀錄。

### 3.3 廢料處理
- [ ] 廢料、包材**每日清運或集中於指定堆置點**,不得堆放於公共區或鄰戶範圍。
- [ ] 依規定分類;有害／特殊廢棄物(舊燈管、化學桶等)依法處理,不得隨意丟棄。
- [ ] 工程完工後,案場必須達到**「可清潔進場」的乾淨度**,不留大型廢料。

## 四、與客戶互動規範(界線要清楚)

- [ ] **不私接**:嚴禁私下接洽屋主承攬本案或衍生工程;所有案件一律經公司。
- [ ] **不議價**:不與客戶談價格、追加、折扣;報價與收款一律由公司負責。
- [ ] **不抱怨**:不在客戶或鄰居面前抱怨公司、抱怨其他工種、抱怨進度或費用。
- [ ] **不亂承諾**:不對客戶承諾工期、材料、變更或保固內容;一律請客戶聯繫公司窗口。
- [ ] **不收受**:不私下收受客戶現金、紅包或回扣;遇情況據實回報。
- [ ] **資訊單一窗口**:客戶的所有需求、變更、疑問,統一回報工務,由公司對外回覆。

> 原則:**客戶我擋,你只管把活做好。** 把界線守乾淨,合作才走得長。

---

## 五、透明施工配合(我們的核心信任機制)

- [ ] **接受全程連線監看**:工地將設置／開放即時影像連線供客戶與公司查看,工班知悉並同意。
- [ ] 鏡頭前後,行為一致——**乾淨、專業、不偷懶**;鏡頭只是把本來就該有的水準呈現出來。
- [ ] 配合公司於關鍵節點拍攝施工照片／影片(隱蔽工程、防水、管線等)留存。
- [ ] 不得遮蔽、移動、關閉或破壞監看設備;設備如有故障立即回報。
- [ ] 透明施工保護的是雙方:**出事找得到責任歸屬,做得好也有紀錄佐證。**

## 六、安全規範

- [ ] 依工項配戴必要防護(安全帽、手套、護目鏡、口罩、防塵具等)。
- [ ] 高處作業確實架設並使用合格設備;用電作業注意漏電與負載。
- [ ] 動火作業(切割、焊接)備妥滅火器材,作業後留置監看至確認無虞。
- [ ] 化學品、油漆、稀釋劑分區存放、保持通風,遠離火源。
- [ ] 工地保持通道淨空與逃生動線;不違規占用消防設施。
- [ ] **意外或損壞第一時間回報工務**,不得隱匿或私下處理;誠實通報者從寬,隱匿者從重。

## 七、違規處置階梯

| 階段 | 適用情形(示例) | 處置 |
|------|------------------|------|
| **第一階:口頭警告** | 輕微整潔不到位、回報不確實、保護未到位等首次發生 | 工務當場糾正並記錄;限期改善 |
| **第二階:書面警告＋扣款** | 重複違規、收尾不確實、噪音／作息超時、造成公共區或鄰戶輕微困擾 | 書面記錄並依約自結算款項中扣款(賠償＋改善成本) |
| **第三階:停止派工** | 屢勸不聽、嚴重損壞、引發社區投訴或客訴 | 暫停派工,完成善後與賠償後另議是否續約 |
| **即刻終止(零容忍)** | 抽菸／亂尿／亂吐、辱罵客戶或鄰居、偷竊、私接客戶、破壞監看設備、隱匿重大事故 | **立即停止派工並終止合作**,損失依約求償 |

> 扣款與賠償之金額、計算與抵扣方式,以《按案結算合作合約》約定為準。
