# 裝修裝潢一站統包 · 業務策略與頭腦風暴

> 行業：**裝修・裝潢一站統包（統包 + 工班閉環 + 交付鍍膜）**
> 核心定位：**「乾淨、透明、不被坑、敢負責」的高端統包——買的人花更多錢，是為了買安心。**
> 配套既有素材：`工班閉環一頁卡.html`（給工班看的閉環海報）

---

## 一、一句話商業邏輯

> **用「全程透明 + 按案結算 + 交付鍍膜保固」做出高端統包，鎖定新家／準新成屋的中高端社區，
> 再用「社區守門人（警衛）轉介系統」當低成本獲客引擎，把一個個大樓變成穩定案源。**

越貴的客人越在意「乾淨、不被坑、找得到人負責」——這正是我們的主場。
客人水準越高 → 交付金額越高 → 養得起好工班 → 品質越好 → 好評轉介越快 → 又帶來更多高價案。這是一個**正循環**。

---

## 二、目標客戶（TA）— 打對人比打很多人重要

| 類型 | 痛點 | 我們為什麼贏 |
|------|------|------------|
| **新婚／第一次入新家** | 想要乾乾淨淨、不留隱患、又怕被坑 | 全透明 + 鍍膜 + 保固，買的是安心 |
| **被家人介紹坑過的人** | 家人覺得好，結果坑更慘 | 一切可查、責任明確 |
| **網路做過功課的年輕人** | 知道有工程蟑螂、卻避不掉 | 時時可查看、按案結算，正好打中他做的功課 |
| **中高端社區住戶** | 要求高、付得起、怕被打擾 | 工地規範（不抽煙、早撤場）正是我們的標配 |

**最高價值、最難打進去**：管理嚴格的高端社區（要求工班社區內不抽煙、兩三點先撤場、不擾鄰）。
他們內部通常已有穩定配合的廠商——**這正是「警衛轉介」要切入的點**，一旦進去，客單價與複購最高。

---

## 三、必殺技：社區守門人轉介系統（警衛公關）

### 做法
鎖定「會待比較久」或「常配合」的社區／大樓，把預算從傳統廣告牌、名片，改成**實用型品牌小物**送給警衛：

- **手機支架** — 印公司名 + 電話 + 業務手機
- **小電風扇** — 印公司名 + 電話 + 業務手機

> **鐵律：公司名稱、電話一定要「非常非常非常」清楚。** 小物的本質是「天天在警衛眼前的廣告牌」。

### 為什麼有效
- 警衛室基本上不是在看劇、就是在吹電扇——**我們送的東西正好是他每天會用、會看的**。
- 打幾聲招呼、混個臉熟，時間久了**一定會有轉介**。住戶問「有沒有認識的師傅」，警衛第一個想到的就是你。
- 這是**信任借力**：警衛的推薦，等於幫你做了第一層背書，成交率遠高於陌生開發。

### 規模化（把單點變系統）
- [ ] 建立**社區名單**：標記「常配合 / 待較久 / 高端」三級，優先攻高端。
- [ ] 每進一個社區 → 警衛室小物 + 業務本人露臉 ≥ 3 次（混熟才有轉介）。
- [ ] 設計**轉介回饋**：警衛介紹成交 → 給明確、體面的謝禮（現金禮券／節慶禮盒），讓他有動力持續介紹。
- [ ] 小物上印**統一業務手機**，方便追蹤哪個社區來的案子（歸因）。
- [ ] 節慶（端午、中秋、過年）再補一輪小物或禮盒，維繫關係。

## 四、核心賣點：全透明化 + 按案結算

兩個一起用，才會形成健康正循環：

1. **全程透明、時時可查看** — 進度、用料、現場狀況隨時看得到。打中「做過功課、怕被坑」的客人。
2. **按案結算，不按天結算**（對屋主、對師傅都一樣）
   - 師傅按案領 → **會盡快完成、不會拖**。
   - 屋主看到的永遠不是偷懶磨時間的狀態 → 信任感拉滿。
   - 對我們：時程可控、口碑好。

> 結果：**屋主滿意 → 好評轉介 → 更高端的客人 → 更高交付金額**，正循環成形。

## 五、痛點地圖（客人躲不掉的坑 → 我們的解法）

| 客人遇到的坑 | 感受 | 我們的對應 |
|------------|------|----------|
| 工程蟑螂、工班削價競爭 | 怕便宜沒好貨 | 統包負責、按案結算、品質掛保證 |
| 出事找不到責任歸屬 | 沒安全感 | 單一窗口統包，責任明確、有保固 |
| 家人介紹反而坑更慘 | 不知該信誰 | 一切透明、可查，不靠人情靠制度 |
| 網路找也是坑 | 做了功課還是踩雷 | 時時可查看，把他的功課變成我們的加分 |
| 師傅到處抽煙、亂尿尿、擾鄰 | 非常厭惡 | 工地規範標配：不抽煙、早撤場、保持乾淨工地 |

> 有的客人**完全不懂**這些坑 → 我們主動講，建立專業形象；
> 有的客人**很懂** → 透明化直接變成超強加分項。兩種人都吃這套。

## 六、利潤引擎：交付即整合「居家鍍膜」

當案子多半是新家／準新成屋，**在交付報價裡整合居家鍍膜**（浴室鍍膜等）：

- **定位**：「我雖然貴一些，但含鍍膜」——把貴講成價值，而不是成本。
- **賣點類比**：像汽車鍍膜在戶外撐兩年，**室內五年都不太會掉**。
- **保固 = 長期關係的鉤子**：有保固就有理由持續跟客人聯絡（居家清潔、定期保養）。
- **複利效應**：有鍍膜 → 之後的清潔案也變輕鬆 → 客人更滿意 → 利潤自然做出來。

> 鍍膜把「一次性裝修」變成「持續性關係」，是高端統包最關鍵的利潤與複購引擎。

## 七、一站統包的閉環（這就是引擎公式）

```
高價案
  → 派給配合規則的好工班（養得起）
    → 透明施工 / 乾淨工地 / 不擾鄰
      → 屋主滿意、好評
        → 轉介 + 高端社區口碑
          → 更多高價案（越轉越快）
```

對工班的核心承諾（呼應 `工班閉環一頁卡.html`）：
> **「你只要顧好品質，案子、收款、客戶我全包。」**
> 接案穩、不被拖款、不用面對奧客、做完不斷線——工班願意守規則，是因為跟我們合作對他最有利。

## 八、落地執行（30 / 60 / 90 天）

**第 1 個月 — 建系統**
- [ ] 列出 10–20 個目標社區，分級（高端優先）
- [ ] 設計＋下單第一批警衛小物（手機支架、小電風扇，名稱電話超清楚）
- [ ] 訂出工地規範 SOP（不抽煙、撤場時間、乾淨工地標準）
- [ ] 確定「按案結算」對屋主／師傅的標準合約版本

**第 2 個月 — 鋪點 + 試打**
- [ ] 警衛室鋪小物 + 業務露臉，每社區 ≥ 3 次
- [ ] 上線「透明進度可查看」機制（群組／相簿／表單，先簡單能跑就好）
- [ ] 把居家鍍膜寫進標準報價選項

**第 3 個月 — 收轉介 + 優化**
- [ ] 追蹤每個社區的轉介來源與成交
- [ ] 建立警衛轉介回饋制度
- [ ] 用前期案例做口碑內容（乾淨工地對比照、屋主好評）

### 建議追蹤的數字（KPI）
- 每個社區的**轉介數 / 成交數**
- **鍍膜整合率**（含鍍膜的案子占比）
- 平均**客單價**、**完工天數**（按案結算後應下降）
- **好評 / 轉介率**

---

## 九、紅線與風險

- ⚠️ 警衛小物與回饋要**體面但合規**，避免變成不當利益輸送的觀感；定位是「品牌曝光 + 人情」。
- ⚠️ 「時時可查看」一旦承諾就要做到，否則反噬信任。
- ⚠️ 「按案結算」要把驗收標準、付款節點寫清楚，避免糾紛。
- ⚠️ 鍍膜的保固年限與條件要白紙黑字，別過度承諾。

## 十、可直接用的話術 / 文案種子

**對屋主（一句話定錨）**
> 「我們全程透明、隨時讓你看，按案結算不拖工，交付還含鍍膜保固——你多花的一點，買的是『乾淨、不被坑、找得到人負責』。」

**對警衛（混熟用）**
> 「大哥這支小電扇／手機架送你用，上面有我電話。住戶要是問起裝修、找師傅，幫我介紹一下，成的話一定好好謝你。」

**對工班（閉環承諾）**
> 「你只要顧好品質，案子、收款、客戶我全包。跟我合作，你接得穩、收得到、不用面對奧客。」
