# 3Q 室內裝修統包 — 內部・客戶旅程營運 SOP

> 文件性質:**內部營運手冊**。
> 核心原則:**全程透明、有人負責到底、按案結算不拖工、一次裝修 → 長期關係。**

> ⚠️ 文中工時、頻率、人力配置為**假設值/建議值,需依實際團隊規模與案量調整**。

## 二、各階段 SOP 明細

### ① 接案 / 引流

| 項目 | 內容 |
|------|------|
| **負責角色** | 業務 / 行銷 / 警衛地推網 |
| **動作清單** | 1. 警衛地推、老客轉介、廣告引流到 LINE OA<br>2. 自動歡迎訊息 + 服務簡介<br>3. 初步需求問答(坪數、屋況、預算帶、時程)<br>4. 判斷分級(基礎/標準/頂級)並排估價 |
| **產出文件** | 客戶基本資料卡(進 CRM)、初步需求紀錄 |
| **客戶觸點** | LINE OA 對話、社區/警衛口碑 |
| **品質關卡** | 是否符合目標客群?是否高端社區?(高價篩選) |
| **自動化 / LINE 觸發** | ✅ 加好友自動歡迎詞;✅ 表單收集需求;✅ 自動建立 CRM 名片 |

### ② 到府估價

| 項目 | 內容 |
|------|------|
| **負責角色** | 估價師 + 專案窗口 |
| **動作清單** | 1. 預約現勘(LINE 自動提醒)<br>2. 現場丈量、拍照、記錄屋況與隱患<br>3. 確認需求與風格、說明透明連線與按案結算制度<br>4. 當場給「分級建議」與報價區間 |
| **產出文件** | 現勘需求單、屋況照片、初步報價區間 |
| **客戶觸點** | 面對面(信任建立關鍵點) |
| **品質關卡** | 屋況隱患是否全數記錄?客戶痛點是否抓到? |
| **自動化 / LINE 觸發** | ✅ 預約前一天自動提醒;✅ 現勘照片回傳客戶相簿 |

### ③ 報價 / 簽約

| 項目 | 內容 |
|------|------|
| **負責角色** | 專案窗口 + 主管覆核 |
| **動作清單** | 1. 出正式三級報價單(鍍膜整合進頂級)<br>2. 說明保固、清潔、按案結算規則<br>3. 簽電子合約、收訂金<br>4. 建立專案、排工班、開透明連線頻道 |
| **產出文件** | 正式報價單、合約、付款排程、施工排程 |
| **客戶觸點** | LINE 推送報價、電子簽約 |
| **品質關卡** | 報價毛利是否達標?工班是否確認可按案完工? |
| **自動化 / LINE 觸發** | ✅ 報價單推送;✅ 訂金請款通知;✅ 簽約完成自動建立施工頻道 |

### ④ 施工(透明連線)

| 項目 | 內容 |
|------|------|
| **負責角色** | 工班 + 工地負責人 + 專案窗口 |
| **動作清單** | 1. 開工通知 + 直播連線啟用<br>2. 每日進度回報(照片+短文)<br>3. 維持乾淨工地、守社區規矩(禁菸、準時撤場)<br>4. 隱蔽工程(水電/防水)拍照存證<br>5. 變更/追加經客戶確認才動工 |
| **產出文件** | 每日進度紀錄、隱蔽工程存證、變更單 |
| **客戶觸點** | 即時直播、每日 LINE 進度 |
| **品質關卡** | 隱蔽工程驗收;階段查核點(泥作/水電/木作/油漆)逐關簽核 |
| **自動化 / LINE 觸發** | ✅ 開工通知;✅ 每日進度自動彙整推送;✅ 階段完成請款通知 |

### ⑤ 交屋 / 鍍膜

| 項目 | 內容 |
|------|------|

| **自動化 / LINE 觸發** | ✅ 保固自動登錄 CRM;✅ 尾款請款;✅ 滿意度問卷;✅ 轉介邀請 |

## 三、透明施工直播 SOP(子章節)

> 透明連線是 3Q 最難被低價同業複製的信任機制 —— 但**做不好會變成負擔與隱私風險**,務必照 SOP。

### 3.1 怎麼連線

| 項目 | 建議做法(假設值,可調整) |
|------|---------------------------|
| 設備 | 工地固定式網路攝影機(主視角)+ 工地負責人手機(機動拍特寫) |
| 平台 | 客戶專屬連結/頻道,**一案一連結、需登入** |
| 權限 | 僅屋主本人 + 同住家人;連結不可外流 |
| 時段 | 施工時段開放即時看;非施工時段顯示靜態畫面或關閉 |

### 3.2 拍什麼

| 必拍 | 說明 |
|------|------|
| 每日整體進度 | 主視角全景,證明有在動工、工地乾淨 |
| 隱蔽工程 | 水電配管、防水、結構 → **逐項拍照存證**(日後爭議關鍵) |
| 階段查核點 | 泥作/水電/木作/油漆每關完成特寫 |
| 變更前後 | 任何追加/變更,前後對照 |

### 3.3 頻率

| 類型 | 頻率(建議) |
|------|-------------|
| 即時直播 | 施工時段全程開放 |
| 進度彙整推送 | **每日 1 次**(下午撤場後),照片+短文 |
| 隱蔽工程存證 | 即時上傳,當天通知客戶查看 |
| 階段查核點 | 每階段完成即時通知 |

### 3.4 隱私與資安注意

| 風險 | 對策 |
|------|------|
| 工人隱私/勞權 | 事前告知工班「會直播」,**鏡頭對工地不對人臉特寫**;禁拍工人用餐/休息 |
| 客戶隱私 | 連結需登入、一案一連結、不公開;**禁止把客戶工地畫面當公開行銷素材**,除非取得書面同意 |
| 資料保存 | 隱蔽工程存證至少保存至保固期滿;影像依個資原則限期清理 |
| 收音 | 預設**不開麥克風收音**,避免錄到私下對話 |
| 斷線/故障 | 設備故障 30 分鐘內通知客戶並補拍,避免「斷線=偷懶」的疑慮 |

> 直播的目的是**證明乾淨、證明用心、留下存證**,不是監控工人。守住這條線,透明才是資產。

## 四、跨階段共用原則

- **單一窗口**:每案一位專案窗口,客戶任何問題只找一個人。
- **按案結算**:對工班、對屋主皆按案不按天,杜絕拖工(見策略全書)。
- **CRM 貫穿**:從①到⑥每個觸點都進 CRM,交屋後自動轉入長期關係流程。
- **缺失零交屋**:點交前缺失全數結清,品質不妥協。
