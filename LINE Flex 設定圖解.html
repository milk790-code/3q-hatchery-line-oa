Flex Message 設定 + 歡迎訊息

從 LINE Official Account Manager 後台，把 Carousel JSON 設為歡迎訊息的完整步驟。

1進入「訊息設定」→「歡迎訊息」

路徑：LINE Official Account Manager › 主頁 › 訊息設定 › 歡迎訊息 › 編輯

2新增訊息 → 選「Flex Message」

點「+ 新增訊息」後，在訊息類型下拉選單選「Flex Message」。

3切換到「JSON」編輯模式

在 Flex Message 編輯器右上角，點「JSON」Tab 切換到 JSON 模式（預設是視覺化模式）。

4（可選）設為關鍵字觸發

如果不只歡迎訊息，還想讓用戶輸入「服務」「介紹」時也觸發 Carousel，在自動回應訊息建立關鍵字規則。

5快速複製 JSON

打開 Welcome Carousel.html，滾動到最底部，點「複製」按鈕，再回到 LINE 後台貼上即可。

Welcome Carousel.html → 頁底 Flex JSON 區塊 開啟 →

注意：LINE 的 Flex Message 歡迎訊息目前每次最多發送 5 個 bubble（已符合我們的 5 張設計）。如果看不到 JSON 模式，確認你的帳號類型是「官方帳號」而非「個人帳號」。
