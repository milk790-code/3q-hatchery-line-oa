附錄 · LINE 專屬規則

FYI
A1

免費版的訊息上限

一般帳號（未驗證）每月有 200 則群發訊息 配額。一對一回覆、自動回覆 不計入，只有「群發訊息」算。

  • 用 carousel push 一次發給所有好友 = 1 則（按好友數計算）
  • 超過 200 則需升級「輕用量方案」NT$800/月，5000 則
  • 初期建議：一週 1 推 carousel，先觀察 open rate
QUOTA
免費200 則 / 月 輕用量5,000 則 · NT$800 中用量15,000 則 · NT$4,000 本月12 / 200 已用
A2

申請藍勾勾驗證帳號

驗證帳號可以在 LINE 內被搜尋（一般帳號只能掃 QR 碼或點短網址加入），對接觸新客很重要。

  • 條件 1：月發訊量穩定達 200 則以上
  • 條件 2：提供 統編 + 公司登記證明
  • 條件 3：實體店面照片 / 官網 URL
  • 路徑：主頁 → 設定 → 帳號設定 → 申請驗證
  • 審核期 5–7 工作天，免費

A4

標籤分類 · 分群推播

好友越多，分群越重要。LINE 後台可以對好友打「標籤」，下次群發只發給某個標籤。

  • 建議標籤：入駐中 · 等待諮詢 · 已成案 · 已合作完
  • 路徑：主頁 → 聊天 → 點對方頭像 → 加標籤
  • 群發時可選「只發給 標籤＝XX 的好友」
