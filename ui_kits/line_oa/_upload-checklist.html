04 / 04

Toggle Off Defaults

關閉 LINE 內建自動回應與歡迎訊息

LINE 預設的「自動回應」跟「歡迎訊息」會搶在 Webhook 之前回覆使用者，蓋掉我們設計好的客製訊息。
兩個都關掉，讓 Cloudflare Worker 接管所有對話。

  1. 左側 → 自動回應 → 頂部 toggle 切「停用」
  2. 左側 → 加入好友的歡迎訊息 → toggle 切「停用」
  3. 部分版本把這兩個放在 設定 → 回應設定 底下，找不到時往那邊翻
  4. 確認頁面狀態顯示「目前狀態：停用」（不是「啟用中」）

打開 LINE app → 搜尋 @121LKSPE → 加好友 → 對照下列檢查：
