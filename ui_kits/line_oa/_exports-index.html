素材匯出工具

每一個 LINE OA 上線需要的圖檔，都有一個對應的匯出頁。 開啟連結 → 按右上「⬇ 下載 PNG」 → 把產出的圖直接上傳到 LINE 後台。 若需要自動觸發下載（給 CI 用），加 ?download=1。 若要乾淨截圖（不顯示工具列、無黑底），加 ?bare=1。
