# Social Extensions UI Kit

Templates for cross-platform extension, using the same anchors as the LINE OA system:
- 7-color palette
- Cormorant Garamond / Noto Serif TC for display
- Gold ≤ 3% as hairlines only
- Locked color-grade LUT across all photography

| File | Surface | Spec |
|---|---|---|
| `IGPost.jsx` | Instagram Feed | 1080×1350 (4:5), top photo / 320px type bar |
| `IGStory.jsx` | IG Story / Threads | 1080×1920, full-bleed photo + bottom 200px paper strip |
| `TikTokCover.jsx` | TikTok / Reels | 1080×1920, title in upper third, photo lower 60% |
| `ThreadsPost.jsx` | Threads post card | 1080×1350, text-driven variant |

`index.html` is a gallery view showing all four templates side-by-side at scaled size, with a TikTok/IG safe-area overlay toggle.
