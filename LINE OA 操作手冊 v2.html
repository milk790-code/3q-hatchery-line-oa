LINE OA SETUP GUIDE · v2

3Q貢丸 · LINE 官方帳號上線手冊

依序完成以下步驟，約 30–60 分鐘可全部上線。已完成的項目打勾追蹤進度。

P0 · 上線前必備
1
上傳 PNG 到 GitHub Pages
將以下 5 張圖片放到你的 GitHub repo 的 assets/exports/ 目錄，push 後自動部署。

2
跑 Rich Menu 上傳腳本
在終端機執行，自動建立三版 Rich Menu 並回傳 3 個 ID。
export CHANNEL_TOKEN="你的Token" && chmod +x export/upload-richmenu.sh && ./export/upload-richmenu.sh
3
填入 worker.js 的 Rich Menu ID
腳本跑完後把三個 ID 填進 worker.js 對應的常數。
const RICHMENU_NEW = "richmenu-xxx";
const RICHMENU_INQUIRED = "richmenu-yyy";
const RICHMENU_CONVERTED = "richmenu-zzz";
4
設定歡迎訊息 + Carousel
在 LINE Official Account Manager 設定歡迎訊息。

路徑：LINE Official Account Manager → 主頁 → 訊息設定 → 歡迎訊息 → 編輯

步驟：

① 點「新增」→ 選「Flex Message」

② 點右上角「JSON」切換編輯模式

③ 貼上 Welcome Carousel.html 頁底的 JSON

④ 點「預覽」確認，再按「儲存」

5
設定 6 組關鍵字自動回覆
路徑：自動回應訊息 → 建立 → 依下表貼入

我想說說我的店 / 我的店 / 我想說 → 好。先告訴我們三件事：1. 你的店在哪裡 2. 你做的東西是什麼 3. 你現在最頭痛的一件事

好物 / 好照 / 拍照 / 500 → 「好物・好照」從一個產品開始，500 元起。回「實例」看三個。

客製 / 行銷 / 陪跑 → 客製行銷是季度規劃。先約 30 分鐘免費諮詢。回「諮詢」開時段。

進度 / 走到哪 → 請告訴我們你的店名 / 報名編號，會回你目前進度。

實例 / 案例 / 看作品 → 本月入駐：01 阿婆ㄟ切仔麵店 02 三代米舖 03 鹿港織坊。回 01/02/03 看。

諮詢 / 預約 → 好。告訴我們：店名 / 角色 / 想聊什麼。24h 內寄時段。

6
QA 測試
加入帳號後逐項點過一次，確認所有按鈕和自動回覆正常。

7

8
会員卡背景
在 worker.js 的 memberCardFlex() 引用 memberBg URL，自動生成個人化會員卡。
